Timestamp Microservice

Description

Add any valid date format onto the end of the url to receive a JSON object containing a JSON unix and regular timestamp.

Usage:

https:///ssc-timestamp.glitch.me/December%2015,%202015

https:///ssc-timestamp.glitch.me/1450137600

Output:

{"unix": 1450137600, "natural": "December 15, 2015"}
